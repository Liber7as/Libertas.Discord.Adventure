# Libertas.Discord.Adventure

Turn-based cooperative Discord RPG bot — multiplayer adventure engine with AI companions, progression, and a test & simulation suite.

## Overview

This repository contains a .NET-based Discord bot and core game logic for cooperative, turn-based adventures. The project includes the bot host, Discord integration, game engine, data access, and unit tests.

## Quick start

Prerequisites:
- .NET 10 SDK

Build:

```
dotnet build Libertas.Discord.Adventure.sln
```

Run the bot (local):

```
dotnet run --project Libertas.Discord.Adventure/Libertas.Discord.Adventure.csproj
```

Run tests:

```
dotnet test Libertas.Discord.Adventure.Core.Tests/Libertas.Discord.Adventure.Core.Tests.csproj
```
